# Steam Price Alert

🎮 Telegram bot for tracking game prices and receiving alerts when discounts reach your target price.

## Planned Features

- Search games via IsThereAnyDeal API
- Track prices with custom targets
- Telegram notifications
- Steam Wishlist import
- Daily deal digests
- Price history charts
- PostgreSQL support
- Docker deployment
- Async architecture with Aiogram 3

## Tech Stack

- Python 3.12
- Aiogram 3
- PostgreSQL
- SQLAlchemy 2
- Alembic
- APScheduler
- aiohttp
- Docker
- pytest
